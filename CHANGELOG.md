# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-07

### Added
- Initial release of SecureVault
- Backend API with Clean Architecture pattern
- AES-256 encryption for notes
- React frontend with TypeScript
- MVVM architecture in frontend
- Full CRUD operations
- GitHub Actions CI/CD pipeline
- Comprehensive documentation

### Security
- AES-256-CBC encryption with unique IVs per note
- PBKDF2 key derivation with 100,000 iterations
- Secure password handling recommendations
