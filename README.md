# SecureVault - Encrypted Note-Taking Application

A secure note-taking application with AES-256 encryption, built with Clean Architecture.

## Architecture
- **Backend**: .NET 8 Web API with Clean Architecture
- **Frontend**: React with TypeScript and MVVM pattern
- **Database**: SQL Server with Entity Framework Core
- **Security**: AES-256-CBC encryption with unique IVs

## Features
- ✅ Encrypted notes (AES-256)
- ✅ Clean Architecture (Domain, Application, Infrastructure, API)
- ✅ MVVM pattern in frontend
- ✅ Full CRUD operations
- ✅ TypeScript for type safety
